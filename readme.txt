La pagina web prende spunto dall'esercizio CSS-3. Per quanto riguarda le caratteristiche
XHTML e CSS, la home page è strutturata come una tabella in cui a sinistra è presente un menù
a lista. 
Abbiamo utilizzato molto gli anchor, in particolare per raggiungere le altre
due nostre pagine web e in generale per altri siti. 
Inoltre, grazie alla pseduo classe hover, quando si è col cursore sul titolo di tali film, appare la rispettiva locandina.
